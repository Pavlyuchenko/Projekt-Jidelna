Formátování textu

Pokud chceš použít formátování textů pro recenzi, hlášku nebo přiznání, musíš použít HTML tagy. Je to opravdu jednoduché. Stačí napsat začáteční tag ve formě <nějaký text>, pak přidat svůj text a ukončit pomocí končícího tagu ve formě </nějaký text>

<b>Tučný text</b> -> Tučný text

<i>Kurzíva</i> -> Kurzíva

<sub>Dolní index</sub> -> Dolní index

<sup>Horní index</sup> -> Horní index

<br />Zalomení řádku ->


<small>Malý text</small> -> Malý text

<big>Velký text</big> -> Velký text
